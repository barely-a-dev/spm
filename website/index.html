SPM is a powerful yet lightweight package manager that simplifies distribution and installation. With support for both local and remote packages, version control, and secure package management, SPM makes it easy to manage your software ecosystem.

Quick Start Guide

Installation
Create Package
Publish

Install a package from the repository:

sudo spm -I package_name

Update package database:

sudo spm -u

Create a package from a directory:

spm -P source_dir output.spm

Create a package from a single file:

spm -f input_file output.spm

Publish a package to the repository:

spm -b package.spm

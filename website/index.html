SPM - Simple Package Manager

Version 1.4.17

SPM is a lightweight, efficient package manager designed for simplicity and ease of use. It supports both local package installation and remote package management through GitHub repositories.

Quick Start

Install a package:

$ sudo spm -I package_name

Create a package from a directory:

$ spm -P source_dir output.spm

Update package database:

$ sudo spm -u
